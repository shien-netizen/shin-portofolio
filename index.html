Payment System

  • Pembayaran menggunakan DP 50% di awal sebelum pengerjaan dimulai.
  • Sisa pembayaran dilakukan setiap 1 artwork selesai.
  • Jika memesan lebih dari 1 artwork, pembayaran tidak digabung.

Contoh: Order 3 Artwork

  • DP dibayar di awal
  • Artwork ke-1 selesai → bayar
  • Artwork ke-2 selesai → bayar
  • Artwork ke-3 selesai → bayar

Sistem ini dibuat supaya proses kerja aman, adil, dan transparan untuk kedua belah pihak.
